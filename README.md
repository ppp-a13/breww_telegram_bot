# BREWW: Telegram Bot для чайного магазина

Учебный проект, сделанный во время изучения Python, БД и разработки ботов.
Бот моделирует работу небольшого чайного магазина: каталог, корзина, покупки, панель администратора.

## Стек разработки
- Python 3.14
- aiogram 3.28
- SQLAlchemy 2.0 (async) + aiosqlite
- Alembic
- python-dotenv

## Функционал
- Каталог товаров по категориям + плагинация
- Карточка товара с изображением
- Корзина: добавление, очистка, просмотр, оформление заказа
- Панель администратора: управление товарами и категориями, поиск пользователей, редактирование карточек
- FSM для многоуровневых сценариев (добавление товара, пополнение баланса)

## Запуск
1. Клонировать репозиторий
2. Создать виртуальное окружение и установить зависимости:
```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
```
3. Создать `.env` на основе `.env.example` и заполнить:
BOT_TOKEN=token_from_@BotFather
ADMIN_IDS=your_telegram_id (допускается несколько значений через запятую)
4. Применить миграции:
```bash
   alembic upgrade head
```
5. Запустить бота:
```bash
   python main.py
```

## Структура

```
├── database/
│   └── models/      # SQLAlchemy-модели таблиц
├── handlers/        # обработчики сообщений, разбиты по модулям
├── keyboards/       # клавиатуры для каждого раздела
├── repositories/    # весь код работы с БД
├── middlewares/     # middlewares для передачи сессии БД в каждый хэндлер
├── migrations/      # Alembic-миграции, история изменений схемы БД
├── filters/         # фильтры: IsAdmin, проверка баланса при покупке
├── states/          # FSM-состояния для многоуровневых сценариев
└── main.py          # главный скрипт: инициализация бота, диспетчера, роутеров
```
